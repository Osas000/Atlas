# Atlas

> **An AI-Powered Personal Operations Platform for Professional Work**

---

# Vision

Atlas is not another AI chatbot.

Atlas is a professional operating platform designed to help an individual discover opportunities, execute high-quality work, continuously learn, and increase legitimate earnings through intelligent assistance.

Atlas combines planning, execution, learning, and operational intelligence into a single integrated workspace.

---

# Mission

The mission of Atlas is simple:

Help the user make better professional decisions every day.

Atlas should:

- Discover legitimate opportunities.
- Help complete professional work.
- Learn from every interaction.
- Improve recommendations over time.
- Increase productivity.
- Increase sustainable earnings.
- Become a trusted long-term professional partner.

---

# Guiding Philosophy

Atlas is built around one principle:

> **Always recommend the highest-value next action.**

Everything in Atlas exists to answer one question:

**What should I do next to move closer to my professional goals?**

---

# Core Principles

Atlas is:

- Local-first
- Privacy-first
- AI-assisted
- Mission-driven
- Modular
- Extensible
- Reliable
- Professional

Atlas is **not**:

- A social network
- A marketplace
- A general-purpose chatbot
- A replacement for human judgment

Atlas augments the user's decision-making rather than replacing it.

---

# Architecture Overview

```text
User
   │

   │

   │

   │
```

Every subsystem communicates through the Operations Core.

The architecture is intentionally modular so that individual components can evolve independently.

# Version One Scope

Version One focuses on creating a stable desktop application that can:

- Organize professional work.
- Discover opportunities.
- Assist with applications.
- Help execute freelance tasks.
- Remember useful information.
- Track progress.
- Learn continuously.

Future releases may expand beyond these capabilities.

---

# Technology Stack

Desktop: Electron

Frontend: React + TypeScript

Build Tool: Vite

Styling: Tailwind CSS

Components: shadcn/ui

State Management: Zustand

Database: SQLite

ORM: Drizzle ORM

Validation: Zod

Logging: Pino

Testing: Vitest + Playwright

Package Manager: pnpm
